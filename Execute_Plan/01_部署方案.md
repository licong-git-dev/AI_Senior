## 一、基础设施采购清单

### 1.1 云服务器

| 阶段 | 配置 | 价格 | 适用场景 |
|------|------|------|---------|
| 起步 | 2核4G + 40G SSD | ¥99-150/月 | 0-500 用户 |
| 增长 | 4核8G + 80G SSD | ¥300-400/月 | 500-2000 用户 |
| 规模 | 8核16G + 160G SSD | ¥700-1000/月 | 2000-10000 用户 |

**购买操作步骤：**
```
1. 打开 https://www.aliyun.com → 注册/登录
2. 搜索 "云服务器ECS" → 选择 "经济型 e 实例"
3. 地域选择：华中（武汉）或华东（上海） — 离目标用户近
4. 镜像选择：Ubuntu 22.04 LTS
5. 带宽选择：5Mbps（按固定带宽计费，比按量便宜）
6. 购买时长：选 1 年有折扣（约 6-7 折）
7. 安全组：开放 22(SSH)、80(HTTP)、443(HTTPS)、8000(API)
```

**省钱技巧：**
- 新用户首购 99 元/年（2核2G），够内测用
- 学生认证 9.9 元/月
- 双 11 / 618 续费最划算，提前囤

### 1.2 域名

```
推荐域名：anxinbao.com 或 anxinbao.cn
购买平台：阿里云万网 / 腾讯云 DNSPOD
价格：¥55/年（.com） 或 ¥29/年（.cn）

注意：.cn 域名需要备案后才能解析到国内服务器
```

**备案流程（必须）：**
```
1. 阿里云控制台 → ICP 备案
2. 准备材料：身份证正反面、域名证书、网站负责人照片
3. 提交审核 → 管局审核（7-20 个工作日）
4. 审核通过后才能将域名解析到国内服务器

小技巧：在等备案期间，可以用 IP 地址先做内测
```

## 三、部署操作（一步一步跟着做）

### 3.1 服务器初始化

SSH 登录服务器后，执行：

```bash
# 1. 系统更新
sudo apt update && sudo apt upgrade -y

# 2. 安装 Docker
curl -fsSL https://get.docker.com | sudo sh
sudo usermod -aG docker $USER

# 3. 安装 Docker Compose
sudo apt install docker-compose-plugin -y

# 4. 安装 Git
sudo apt install git -y

# 5. 配置防火墙
sudo ufw allow 22/tcp    # SSH
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
sudo ufw enable

# 6. 创建项目目录
mkdir -p /opt/anxinbao
cd /opt/anxinbao
```

### 3.2 拉取代码并配置

```bash
# 1. 克隆代码
git clone <你的仓库地址> .

# 2. 复制并编辑环境变量
cp .env.example .env
nano .env
```

**`.env` 必须修改的配置：**
```bash
# ========= 必改 =========
DEBUG=false
DATABASE_URL=postgresql://anxinbao:你的数据库密码@db:5432/anxinbao
DB_PASSWORD=你的数据库密码          # 用于 docker-compose
REDIS_URL=redis://redis:6379/0

# JWT 密钥（必须替换！用下面的命令生成）
# python3 -c "import secrets; print(secrets.token_urlsafe(48))"
JWT_SECRET_KEY=替换为随机字符串

# AI 服务
DASHSCOPE_API_KEY=你的灵积API密钥

# 科大讯飞
XFYUN_APPID=你的AppID
XFYUN_API_KEY=你的APIKey
XFYUN_API_SECRET=你的APISecret

# CORS（替换为你的域名）
ALLOWED_ORIGINS=https://anxinbao.com,https://www.anxinbao.com

# ========= 可选 =========
SMS_ACCESS_KEY_ID=短信服务密钥
SMS_ACCESS_KEY_SECRET=短信服务密钥
```

### 3.3 SSL 证书配置

```bash
# 安装 certbot
sudo apt install certbot -y

# 申请证书（需要域名已解析到服务器 IP）
sudo certbot certonly --standalone -d anxinbao.com -d www.anxinbao.com

# 证书位置（配置到 nginx）
sudo mkdir -p /opt/anxinbao/ssl
sudo cp /etc/letsencrypt/live/anxinbao.com/fullchain.pem /opt/anxinbao/ssl/
sudo cp /etc/letsencrypt/live/anxinbao.com/privkey.pem /opt/anxinbao/ssl/

# 设置自动续期（crontab）
echo "0 3 1 * * certbot renew --quiet && cp /etc/letsencrypt/live/anxinbao.com/*.pem /opt/anxinbao/ssl/ && docker restart anxinbao-nginx" | sudo tee -a /etc/crontab
```

### 3.4 启动服务

```bash
cd /opt/anxinbao

# 开发/内测阶段（不启动 Nginx，直接用 8000 端口）
docker compose up -d

# 生产环境（启动 Nginx 反向代理 + SSL）
docker compose --profile production up -d

# 查看运行状态
docker compose ps

# 查看日志
docker compose logs -f api     # API 日志
docker compose logs -f db      # 数据库日志
```

### 3.5 验证部署

```bash
# 1. 健康检查
curl http://localhost:8000/health
# 预期返回: {"status":"ok","service":"anxinbao-server","version":"1.0.0"}

# 2. API 文档
# 浏览器访问: http://你的IP:8000/docs

# 3. 数据库检查
docker exec anxinbao-db psql -U anxinbao -c "SELECT count(*) FROM information_schema.tables WHERE table_schema='public';"

# 4. Redis 检查
docker exec anxinbao-redis redis-cli ping
# 预期返回: PONG
```

### 3.6 前端部署

```bash
# 方案一：与后端同域名部署（推荐，简单）
# 在本地构建前端
cd anxinbao-pwa
npm run build

# 将 dist 目录内容上传到服务器的 static 目录
scp -r dist/* user@服务器IP:/opt/anxinbao/static/

# 方案二：独立部署到 CDN（性能更好）
# 推荐使用 Vercel / Netlify / 阿里云 OSS + CDN
# Vercel 部署（最简单）：
# 1. 把 anxinbao-pwa 推到 GitHub
# 2. 在 vercel.com 导入仓库
# 3. 自动构建部署，绑定自定义域名

# 方案三：腾讯云静态网站托管（备案域名可用）
# 1. 登录腾讯云 → 静态网站托管
# 2. 上传 dist 目录
# 3. 绑定域名 app.anxinbao.com
```

## 五、支付接入（第二阶段）

### 微信支付
```
1. 注册微信商户号 https://pay.weixin.qq.com
2. 提交材料：营业执照、法人身份证、银行账户
3. 审核通过后获取：商户号、API密钥、证书
4. 对接 app/api/payment.py（已预留接口）
```

### 支付宝
```
1. 注册支付宝开放平台 https://open.alipay.com
2. 创建应用 → 签约当面付/手机网站支付
3. 获取 APPID、私钥、支付宝公钥
```

**个人开发者替代方案：**
```
如果暂时没有营业执照，可以先用：
- 微信/支付宝收款码（手动确认）
- 赞赏码（适合内测期）
- 注册个体工商户（最快 1 天，几百元）
```

## 六、数据库初始化

首次部署时，Docker Compose 会自动创建数据库表。如需手动初始化：

```bash
# 进入 API 容器
docker exec -it anxinbao-api bash

# 运行数据库初始化
python -c "from app.models.database import init_db; init_db()"

# 创建管理员账户（可选）
python -c "
from app.models.database import SessionLocal, UserAuth
from app.core.security import get_password_hash
db = SessionLocal()
admin = UserAuth(
    username='admin',
    password_hash=get_password_hash('你的管理员密码'),
    role='admin',
    is_active=True
)
db.add(admin)
db.commit()
print('管理员创建成功')
"
```
