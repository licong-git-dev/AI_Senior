# 安心宝 — 推广方案

> 核心策略：先打情感牌引发共鸣，再用产品体验形成口碑

---

## 一、推广三阶段

```
第一阶段（0-1个月）        第二阶段（1-3个月）         第三阶段（3-6个月）
种子用户 + 产品打磨        内容营销 + 精准投放          口碑裂变 + 城市扩展

目标：50 个真实用户        目标：1000 付费用户          目标：5000+ 付费用户
预算：¥0（纯靠人情）       预算：¥5000-10000/月        预算：¥20000-50000/月
```

---

## 二、第一阶段：种子用户（前 4 周）

### 2.1 从身边开始（第 1 周）

**目标：找到 20 个真实使用的家庭**

```
渠道一：自己的朋友圈
  发一条真诚的朋友圈：
  ——————————————————
  做了一个小产品，想请大家帮忙试试。

  如果你的爸妈/爷爷奶奶一个人在家，
  安心宝可以用方言陪他们聊天，
  每天给你推一份"今日爸妈"日报，
  让你知道他们今天过得怎么样。

  武汉话、鄂州话、普通话都能聊。
  完全免费，只求真实反馈。

  有兴趣的扣 1，我拉你进群。
  ——————————————————

渠道二：家人群/亲戚群
  让爸妈先用起来，拍使用视频发家族群
  真实场景 > 一切广告

渠道三：同事/同学群
  "有没有父母独居的朋友？帮我测个产品"
  25-40岁的人群转化率最高
```

### 2.2 建立内测群（第 2 周）

```
微信群名：安心宝内测家族群
群规模：30-50 人（20 个家庭 + 团队成员）

每日运营动作：
  早上 9:00  — 发一个老人使用安心宝的暖心截图
  中午 12:00 — 问大家昨天爸妈的安心日报分数怎么样
  晚上 20:00 — 收集 Bug 和建议

关键动作：
  ✅ 每个 Bug 24 小时内修复并回复
  ✅ 用户提的好建议，1 周内上线，@感谢他
  ✅ 收集 5 个"爸妈第一次听到方言时的反应"视频
```

### 2.3 打磨核心体验（第 3-4 周）

```
根据内测反馈重点优化：
1. 方言识别准确率 → 补充常见方言词
2. 问候语是否自然 → 调整模板措辞
3. 安心日报是否有价值 → 调整摘要内容
4. 老人是否愿意说话 → 优化主动问候时机
```

## 三、第二阶段：内容营销（1-3 个月）

### 3.1 短视频矩阵（核心渠道）

**抖音 + 小红书 + 视频号，三端同步发**

#### 内容系列一：「爸妈听到方言的瞬间」（情感爆款）

```
脚本模板：
┌─────────────────────────────────────────┐
│ [开头 3 秒 — 悬念钩子]                     │
│ "我妈一个人在家半年了，                      │
│  直到我给她装了这个东西……"                   │
│                                          │
│ [中间 10 秒 — 核心画面]                     │
│ 老人坐在沙发上，安心宝突然用武汉话说：         │
│ "张妈妈，今天的晚饭恰了么？"                  │
│ 老人惊讶 → 微笑 → 开始聊天                  │
│                                          │
│ [结尾 5 秒 — 情感升华]                      │
│ 子女手机收到推送："妈妈今天很开心"             │
│ 旁白："你不在的时候，安心宝陪着她"             │
└─────────────────────────────────────────┘

拍摄要求：
  - 必须用真实老人（自己爸妈/亲戚）
  - 不要摆拍，抓拍真实反应
  - 竖屏，手机拍即可
  - 加字幕（方言要翻译）
```

#### 内容系列二：「今天爸妈过得怎么样」（日常种草）

```
形式：子女视角，每天晒安心日报截图
文案示例：
  "早上通勤在地铁上看了一眼安心宝，
   妈妈昨天安心指数 9 分，
   和邻居王阿姨聊了广场舞，
   血压 130/82 正常，按时吃药了。
   看完心里踏实多了。"

发布节奏：每天 1 条（可以用不同用户的真实数据）
```

#### 内容系列三：「远方的牵挂」（情感共鸣）

```
形式：故事类短视频，2-3 分钟
选题：
  - "凌晨 3 点，妈妈说头晕"（紧急场景）
  - "爸妈嘴上说挺好，安心日报说他一天只说了 3 句话"
  - "母亲节那天，给妈妈装了安心宝"
  - "86 岁奶奶第一次跟 AI 用方言聊天"
```

**标签矩阵：**
```
核心标签：#安心宝 #方言AI #给爸妈的礼物
情感标签：#远方的牵挂 #思乡 #爸妈 #孝心
场景标签：#独居老人 #空巢老人 #子女关怀
话题标签：#你有多久没给爸妈打电话了 #AI陪伴
```

### 3.2 小红书深度种草

```
账号定位：「安心宝｜让远方的子女更安心」

内容类型：
  1. 产品测评笔记（图文）
     "测了一个月，这个方言AI真的让我妈开心了"

  2. 对比测评
     "给爸妈买了智能音箱 vs 安心宝，差别太大了"

  3. 情感笔记
     "看到安心日报里妈妈说'想儿子了'，我哭了"

  4. 实用攻略
     "独居老人安全清单，最后一个很多人不知道"

发布频率：每周 3-5 篇
```

### 3.3 微信公众号

```
公众号名：安心宝（服务号）

核心功能：
  - 绑定后每日推送安心日报
  - SOS 紧急通知推送通道
  - 菜单栏直达子女端面板

内容策略：
  每周 2 篇推文：
  周二：产品更新/功能介绍
  周五：用户故事/暖心案例
```

### 3.4 朋友圈精准广告

```
投放平台：微信广告（腾讯广告）

人群定向：
  - 年龄：25-45 岁
  - 地域：父母在湖北的用户（一线城市为主）
  - 兴趣：亲子、养老、健康
  - 行为：近期搜索过"老人""独居""养老"

广告形式：朋友圈图片/视频广告
文案测试方案（A/B测试）：
  A: "你的妈妈今天和谁说话了？"
  B: "一杯咖啡的钱，换爸妈一整天的陪伴"
  C: "会说武汉话的AI，让妈妈不再孤单"

预算：¥100/天 起步测试，ROI 达标后放量
目标 CPA：¥30-50/注册用户
```

## 七、转化漏斗目标

```
曝光 → 点击 → 注册 → 激活 → 付费 → 续费

具体指标：
  曝光   100000 人/月
    ↓ 点击率 3%
  点击   3000 人/月
    ↓ 注册率 30%
  注册   900 人/月
    ↓ 激活率 60%（老人端首次对话成功）
  激活   540 人/月
    ↓ 免费→付费转化 10%
  付费   54 人/月
    ↓ 续费率 70%
  续费   38 人/月

ARPU：¥35/月（加权平均）
月收入：54 × ¥35 = ¥1890/月（第3个月预期）
```
