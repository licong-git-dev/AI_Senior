## Day 2 — 服务器初始化

- [ ] **SSH 登录服务器**
  ```bash
  ssh root@你的服务器IP
  ```

- [ ] **安装 Docker**
  ```bash
  curl -fsSL https://get.docker.com | sudo sh
  sudo usermod -aG docker $USER
  sudo apt install docker-compose-plugin -y
  ```

- [ ] **安装 Git 并克隆代码**
  ```bash
  sudo apt install git -y
  mkdir -p /opt/anxinbao && cd /opt/anxinbao
  git clone <你的仓库地址> .
  ```

- [ ] **配置 .env 文件**
  ```bash
  cp .env.example .env
  nano .env
  # 填入 Day 1 获取的所有 API Key
  ```

- [ ] **测试 AI API**
  ```bash
  # 确认灵积 API 可用
  curl -X POST https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation \
    -H "Authorization: Bearer 你的API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"model":"qwen-turbo","input":{"messages":[{"role":"user","content":"你好"}]}}'
  ```

## Day 3 — 部署启动

- [ ] **启动所有服务**
  ```bash
  cd /opt/anxinbao
  docker compose up -d
  docker compose ps    # 确认全部 healthy
  ```

- [ ] **验证部署**
  ```bash
  curl http://localhost:8000/health
  # 浏览器访问 http://你的IP:8000/docs
  ```

- [ ] **创建管理员账户**
  ```bash
  docker exec -it anxinbao-api python -c "
  from app.models.database import SessionLocal, UserAuth
  from app.core.security import get_password_hash
  db = SessionLocal()
  admin = UserAuth(username='admin', password_hash=get_password_hash('你的密码'), role='admin', is_active=True)
  db.add(admin)
  db.commit()
  print('Done')
  "
  ```

- [ ] **全流程测试**
  ```
  1. 浏览器打开 http://你的IP:8000/docs
  2. 测试 /api/auth/register → 注册用户
  3. 测试 /api/auth/login → 获取 token
  4. 测试 /api/chat/send → 发送对话
  5. 确认 AI 正确回复
  ```

## Day 4 — 前端部署 + 备份

- [ ] **构建前端**
  ```bash
  # 在本地
  cd anxinbao-pwa
  npm run build
  # 上传到服务器
  scp -r dist/* user@服务器IP:/opt/anxinbao/static/
  ```

- [ ] **配置 SSL（如果域名已解析）**
  ```bash
  sudo apt install certbot -y
  sudo certbot certonly --standalone -d anxinbao.com
  ```

- [ ] **配置自动备份**
  ```bash
  # 复制 backup.sh 到服务器
  chmod +x /opt/anxinbao/backup.sh
  # 添加定时任务
  crontab -e
  # 添加: 0 3 * * * /opt/anxinbao/backup.sh >> /opt/anxinbao/logs/backup.log 2>&1
  ```

- [ ] **配置监控**
  ```
  注册 UptimeRobot → 添加 http://你的IP:8000/health
  ```
