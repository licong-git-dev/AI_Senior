# 数字生命陪伴 · 成本模型

> 配套 [`DIGITAL_COMPANION_RFC.md`](DIGITAL_COMPANION_RFC.md) 的财务测算。
> 决定"数字生命模式 ROI 是否成立"的关键文档。

## 0. 一句话结论

**Phase 1+2 单老人成本 ≈ ¥38/月**，可由当前 ¥69/月 基础订阅覆盖。
**Phase 4 完整体 ≈ ¥98-168/月**，需 **VIP 版 ¥99-149/月** 或 **B2G 政府兜底** 才能持续。

商业模式不到位 → **不要做 Phase 3-4**。

## 5. 商业模式适配

### 5.1 三档套餐建议

| 套餐 | 月费 | 包含模式 | 目标用户 |
|---|---|---|---|
| **基础版** | ¥29/月 | 当前工具版 | 价格敏感、试用 |
| **陪伴版** | ¥69/月 | + Phase 1+2 | 主流、需要"它记得我" |
| **VIP 版** | ¥149/月 | + Phase 3+4 (qwen) | 高净值、子女在外地 |
| **VIP+ 版** | ¥299/月 | + Claude Sonnet | 最高品质、企业 / B2G |

### 5.2 B2G / B2B2C 路径

- **江岸民政局采购**：单社区 ¥10000-30000/年 = 100 老人 × ¥100/年
  - 民政局补贴一部分基础月费，企业承担陪伴版成本
  - 关键：把 Phase 4 的"健康告警 + 紧急救助"对齐"政府智慧养老 KPI"
- **保险公司协销**：泰康 / 中宏的养老险增值服务
  - 每签一单送 6-12 个月陪伴版
  - 摊销给保险公司，单老人成本 0

### 5.3 不该做的事

❌ 把 Phase 4 强加给 ¥29/月用户（亏本）
❌ 用 Claude Sonnet 全量服务（除非 VIP+ 卖得动）
❌ 推免费版 + 内置广告（伦理风险，老人最反感）

## 7. 何时停止扩展

每季度评估：
- 单 ARPU 成本 > ¥150 且付费率 < 20% → 砍 Phase 4
- 单 ARPU 成本 > ¥80 且 NPS 增幅 < 10 → 砍 Phase 3
- 任意阶段出现严重安全事故 → 全部回滚到工具版

## 8. 当前建议

**先做 Phase 1+2 (qwen-plus)** —— ¥19/月成本由现有 ¥29/¥69 可覆盖，是 ROI 最优起点。

Phase 3+4 等以下条件满足再启动：
- 陪伴版有 ≥1000 付费用户（验证人格 + 记忆的真实价值）
- 商业模式确认 VIP 版能卖（≥10% 用户升 VIP）
- 单老人月成本预测可控在 ¥80 以内
