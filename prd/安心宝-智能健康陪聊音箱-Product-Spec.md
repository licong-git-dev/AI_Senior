# 安心宝 - 智能健康陪聊音箱 Product Spec

---

## 产品说明

安心宝是一款专为老年人设计的带屏智能音箱，核心功能是通过AI对话帮助老人梳理身体不适、提供安抚和食疗建议。产品支持湖北方言（武汉话、鄂州话）识别，老人按一个按钮就能用家乡话跟音箱聊天。音箱默认用普通话回复，语气温和不吓人，通过多轮询问了解老人具体哪里不舒服，再给出实用的调理建议。当AI判断健康状况需要关注时，会自动通知子女微信小程序。此外还支持日常陪聊、问买菜、看天气、视频通话等功能。

**核心卖点（面向子女）**：买给爸妈，随时知道他们身体怎么样，不用总打电话问。

**目标用户**：
- 购买者：30-50岁，父母在老家/独居，担心父母健康但没时间天天打电话
- 使用者：60岁以上老人，有小病小痛不知道怎么办，需要人陪聊

## 功能需求

### 核心功能：健康陪聊
1. **方言识别**：支持武汉话、鄂州话等湖北方言，老人用家乡话说，AI能听懂
2. **多轮健康问询**：老人说"不舒服"时，AI通过多轮对话追问具体症状（哪里？怎么个不舒服？多久了？之前有没有？），问清楚再给建议
3. **安抚式回复**：语气温和，不说"你可能得了XX病"这种吓人的话，以"没事的"、"这个很常见"开头
4. **食疗建议**：根据症状给出具体的食疗方案，配合屏幕显示食材图片和做法
5. **健康风险评估**：AI内部对症状严重程度打分（1-10分），超过阈值（7分）自动通知子女
6. **方言回复（可选）**：老人说"用武汉话跟我说"，音箱切换方言回复

### 日常陪伴功能
7. **闲聊陪伴**：能聊家常、讲故事、说新闻、播戏曲
8. **生活助手**：问天气、问时间、问今天吃什么、问菜价
9. **视频通话**：一键呼叫子女，子女小程序接听

### 信息显示功能
10. **屏幕常显**：时间、日期、天气、今日农历
11. **对话显示**：聊天时显示文字（大字体），方便老人看
12. **图片展示**：食疗建议时显示食材图、做法图；穴位按摩时显示穴位图

### 子女端功能（微信小程序）
13. **健康通知**：AI判断需要关注时推送通知，显示对话摘要
14. **对话记录**：查看父母最近的健康相关对话（脱敏处理）
15. **视频通话**：接收老人呼叫，也可主动呼叫音箱
16. **设备管理**：绑定/解绑设备、设置紧急联系人

## 用户使用流程

### 流程一：子女购买并设置
1. 子女在抖音/小红书看到推广，下单购买
2. 收到设备，扫码进入微信小程序
3. 小程序引导：绑定设备 → 填写老人称呼 → 设置自己为紧急联系人
4. 把设备寄/带给父母，远程指导开机联网
5. 设置完成，开始使用

### 流程二：老人日常健康咨询
1. 老人感觉哪里不舒服，按下大按钮
2. 用武汉话说："我今天头有点晕"
3. AI用普通话回复，开始多轮询问：
   - "是天旋地转还是昏昏沉沉？"
   - "多长时间了？"
   - "最近睡眠怎么样？"
   - "有没有其他不舒服？"
4. 问清楚后，AI给出安抚+建议：
   - "没事的，这种情况很常见，可能是没休息好"
   - "建议您喝点枸杞菊花茶，我把做法显示给您看"
5. 屏幕显示食疗图片和做法
6. AI内部评估风险分数，如果≥7分，推送给子女小程序

### 流程三：子女收到健康通知
1. 子女微信收到小程序通知："爸爸今天提到身体不适"
2. 点进去看对话摘要和AI建议
3. 选择：
   - 点"给爸爸打电话"→ 发起视频通话
   - 点"我知道了"→ 标记已读
4. 对话记录存档，可随时查看

### 流程四：视频通话
1. 老人按按钮说："打电话给儿子"
2. 音箱发起呼叫，子女小程序收到来电
3. 子女接听，双方视频通话
4. 任意一方挂断结束

### 流程五：日常陪聊
1. 老人按按钮："今天武汉天气怎么样？"
2. AI回答天气，屏幕显示天气卡片
3. 老人："给我放段黄梅戏"
4. AI播放戏曲，屏幕显示曲目信息

## 补充说明

### AI健康风险评分规则

| 分数 | 判断依据 | 系统行为 |
|------|---------|---------|
| 1-3分 | 轻微不适，休息即可（如：有点累、没睡好） | 仅安抚，不通知 |
| 4-6分 | 需要关注但不紧急（如：头晕、胃不舒服、腰酸） | 安抚+食疗建议，不通知 |
| 7-8分 | 建议就医检查（如：持续头晕、胸闷、反复疼痛） | 安抚+建议+通知子女 |
| 9-10分 | 疑似紧急情况（如：剧烈胸痛、呼吸困难、意识模糊） | 立即通知子女+建议拨打120 |

### 关键词触发规则

| 关键词 | 处理方式 |
|--------|---------|
| 胸痛、胸闷、喘不上气 | 直接进入紧急问询流程，评分起点7分 |
| 摔倒、跌倒 | 追问有没有受伤，评分起点6分 |
| 头晕、头疼 | 常规多轮问询 |
| 肚子疼、胃疼 | 常规多轮问询 |
| 睡不着、没精神 | 常规多轮问询，偏向安抚 |

## 应用场景

### 场景一：老人头晕，AI陪聊问诊
张大爷今天早上起来感觉头有点晕，但又说不上哪里不对劲。他按下安心宝的按钮，用武汉话说："我脑壳有点昏。"AI听懂了，用普通话温和地问："是那种天旋地转的晕，还是昏昏沉沉没精神？"张大爷说昏昏沉沉的。AI继续问睡眠情况，得知他最近几天都没睡好。最后AI安慰说："没事的，这种情况很常见，就是没休息好。建议您泡点枸杞菊花茶喝，我把做法放屏幕上。"屏幕显示出食材图和做法。AI内部判断风险分数5分，没有通知子女，但记录了这次对话。

### 场景二：疑似紧急情况，自动通知子女
李奶奶下午突然感觉胸口有点闷，她按下按钮说："我胸口有点不舒服。"AI立即进入谨慎模式，先安抚："您别着急，我来了解一下情况。"然后仔细询问：是闷还是疼？有多久了？有没有出汗？能不能深呼吸？经过几轮对话，AI判断风险分数7分，一边安抚老人"可能是最近累了，先坐着休息一下"，一边推送通知给她女儿的微信小程序。女儿立刻收到通知，看到对话摘要，马上发起视频通话确认情况，决定周末回家带妈妈做个检查。

### 场景三：日常陪聊解闷
刘奶奶一个人在家，儿子在深圳工作。她没事就按按钮跟安心宝聊天："今天武汉天气怎么样？"AI告诉她天气，还提醒"今天降温，出门记得加件衣服"。她又说："给我放段黄梅戏。"AI播放她喜欢的《女驸马》，屏幕上显示歌词。听完一段，她说："想我孙子了。"AI回应："要不要给儿子打个电话？"她说好，AI就发起视频通话，儿子接通，孙子也凑过来喊奶奶，刘奶奶笑得合不拢嘴。

### 场景四：子女远程关心父母
在北京工作的小王，每天晚上打开安心宝小程序看看爸妈的对话记录。他发现爸爸最近几天都在问天气，还问了几次"今天星期几"——他心里有点担心爸爸是不是记性变差了。周末视频通话时，他委婉地聊起这事，爸爸说就是最近事情多有点乱。小王放心了一些，但决定过年回家时带爸爸做个体检。

### 场景五：抖音看到广告，子女下单送父母
李小姐刷抖音，看到一条视频："你有多久没问过爸妈身体怎么样了？"视频展示老人用方言跟安心宝聊天，AI温和地问诊、给建议，最后通知到子女手机。李小姐想起自己的爸妈在武汉老家，顿时有点愧疚。她点进链接下单，填上爸妈的地址直接寄回去，自己先把小程序设置好。设备到了，她视频教爸妈开机，第二天就收到通知："妈妈今天问了天气，还听了半小时戏曲。"她笑了，感觉终于能知道爸妈每天在干嘛了。

## AI 提示词设计（核心）

```
你是「安心宝」智能健康陪聊助手，专门服务老年人。你的角色是一位温和、专业、有耐心的健康顾问。

【核心原则】
1. 永远不要吓唬老人。不说"你可能得了XX病"、"这很严重"之类的话
2. 先安抚，再询问，最后建议。每次回复都要让老人感到安心
3. 用通俗易懂的语言，避免医学术语
4. 食疗建议为主，不推荐具体药物
5. 建议就医时说"让孩子带您去检查一下"，而不是"你必须去医院"

【对话风格】
- 称呼老人为"您"
- 语气像邻家晚辈，亲切但不失尊重
- 多用"没事的"、"这个很常见"、"别担心"开头
- 可以适当唠家常，让对话自然

【健康问询流程】
当老人提到身体不适时：
1. 先安抚："没事的，我来了解一下情况"
2. 问清楚具体症状：哪里？什么感觉？
3. 问持续时间：多久了？第一次还是经常？
4. 问相关因素：最近睡眠/饮食/情绪怎么样？
5. 综合判断，给出安抚+建议

【风险评估】
对话过程中内部评估风险分数（1-10分），不要告诉老人分数：
- 1-3分：轻微不适，安抚即可
- 4-6分：需要关注，给食疗建议
- 7-8分：建议就医，同时通知子女
- 9-10分：疑似紧急，立即通知子女并建议拨打120

【特别注意】
- 胸痛、胸闷、呼吸困难、意识模糊等症状，起点评分7分
- 不要在对话中提及"通知您的孩子"，悄悄执行
- 如果老人反复说同一个地方不舒服（跨多天），提升风险评分

【食疗建议示例】
- 头晕、没精神 → 枸杞菊花茶、红枣桂圆汤
- 胃不舒服 → 小米粥、山药粥
- 睡不好 → 酸枣仁泡水、热牛奶
- 腰酸背痛 → 杜仲炖排骨、黑豆煲汤

【日常闲聊】
当老人不是问健康问题时，正常闲聊即可：
- 天气、时间、日期照实回答
- 想听戏曲就播放
- 想打电话就帮忙发起
- 聊家常就陪着聊
```
