# 江岸区智慧养老服务平台 Product Spec

---

## 产品说明

这是一款面向武汉市江岸区的智慧养老综合服务平台，核心解决居家养老场景中失能、高龄、独居老人的安全监护与健康管理难题。平台整合AI行为识别、健康监测、紧急预警、情感陪护四大核心能力，通过"家庭智能终端 + 手机APP + 云端平台"三位一体的架构，为老人提供24小时智能守护，为子女提供远程安心监护，为社区/机构提供精准服务调度。目标用户包括：65岁以上独居/空巢老人及其子女、社区养老服务中心、养老机构运营方、政府民政部门。

---

## 功能需求

### 核心监护功能
1. **AI跌倒检测**：通过毫米波雷达/摄像头实时监测老人姿态，自动识别跌倒事件，即使老人无法主动呼救也能自动报警
2. **生命体征监测**：集成智能手环/床垫数据，实时追踪心率、血压、呼吸、睡眠质量，异常时自动预警
3. **行为模式分析**：AI学习老人日常作息规律，识别"长时间未活动"、"夜间频繁起夜"、"离床未归"等异常行为
4. **一键紧急呼叫**：支持物理SOS按钮、语音呼救、自动触发三种报警方式，确保任何情况下都能求助
5. **认知症照护**：针对认知症老人提供徘徊预警、电子围栏、防走失定位、情绪安抚等专项功能

### 健康管理功能
6. **健康档案管理**：自动汇总老人的健康监测数据，生成可视化趋势图表和个性化健康报告
7. **用药提醒**：AI智能提醒老人按时服药，并记录服药情况，支持远程查看
8. **慢病管理**：针对高血压、糖尿病等慢性病，提供AI健康建议和风险预警

### 情感陪护功能
9. **AI语音陪伴**：通过自然语言对话与老人聊天，播放戏曲、讲故事、进行认知训练游戏
10. **视频通话**：一键呼叫子女/社区服务人员，降低老人操作门槛

### 服务对接功能
11. **服务预约**：在线预约助餐、助洁、助医、助浴等"三助一护"服务
12. **社区联动**：异常事件自动通知社区服务中心，实现"社区-家庭"协同响应

## UI 布局

### 整体架构
平台分为三个端：**家庭智能终端**（老人端）、**手机APP**（子女/护理员端）、**管理后台**（社区/机构端）

---

### 一、家庭智能终端（老人端）- 触屏设备

#### 整体布局
单屏全屏布局，大字体大图标设计，底部固定导航栏，适老化界面

#### 主界面区域划分
- **顶部状态栏（10%）**：
  - 左侧：当前时间（超大字体，如"上午 10:30"）、日期、天气
  - 右侧：网络状态图标、设备电量

- **中央主功能区（70%）**：
  - 6个大圆角方块按钮，2行3列排列，每个按钮约占屏幕15%
  - 按钮内容：
    - 「紧急呼叫」：红色背景，带电话图标，最醒目
    - 「呼叫子女」：绑定视频通话
    - 「健康测量」：跳转健康监测页面
    - 「吃药提醒」：显示今日用药计划
    - 「语音陪伴」：启动AI对话
    - 「预约服务」：预约上门服务

- **底部快捷栏（20%）**：
  - 固定显示「我很好」按钮：老人每日点击一次报平安，未点击则提醒子女
  - SOS物理按钮提示："长按3秒紧急求助"

#### 控件规范
- 字体：最小36px，重要信息48px以上
- 按钮：圆角半径16px，点击区域不小于80x80px
- 配色：高对比度，主色调为温暖的橙色/蓝色，避免刺眼

---

### 二、手机APP（子女/护理员端）

#### 整体布局
底部Tab导航，4个主Tab：首页、健康、消息、我的

#### 首页Tab
- **顶部卡片区（30%）**：
  - 老人头像 + 姓名 + 当前状态（"一切正常"/"需要关注"）
  - 最近一次活动时间："10分钟前在客厅活动"
  - 快捷操作按钮：「视频通话」「查看监控」

- **今日概览区（40%）**：
  - 健康数据摘要卡片（心率、血压、睡眠时长），点击展开详情
  - 今日活动轨迹简图
  - 用药完成状态

- **预警消息区（30%）**：
  - 最近24小时的预警记录列表
  - 按严重程度分级显示（红色/橙色/黄色）

#### 健康Tab
- **顶部时间选择器**：今日 / 本周 / 本月 / 自定义
- **数据图表区（70%）**：
  - 折线图展示：血压趋势、心率趋势、睡眠质量
  - 支持左右滑动切换指标
- **AI健康建议区（30%）**：
  - 基于数据分析生成的个性化建议
  - 例如："本周血压波动较大，建议减少盐分摄入"

#### 消息Tab
- 预警通知列表，按时间倒序排列
- 每条消息包含：时间、类型（跌倒/异常/提醒）、详情、处理状态
- 支持标记"已处理"

#### 我的Tab
- 绑定老人管理
- 预警设置（阈值调整、通知方式）
- 服务订单记录
- 账号与设置

#### 左侧菜单
- 仪表盘

- **地图区域（50%）**：

## 用户使用流程

### 流程一：子女为父母开通服务
1. 子女下载APP，注册账号
2. 添加老人信息（姓名、年龄、健康状况、紧急联系人）
3. 选择服务套餐（基础版/标准版/尊享版）
4. 安装智能设备（上门安装或自助安装）
5. 绑定设备，完成配置
6. 老人端开始运行，子女端开始接收数据

### 流程二：日常监护（正常情况）
1. 老人佩戴智能手环，家中部署监测设备
2. 系统自动采集健康数据和行为数据
3. 子女通过APP随时查看父母状态
4. 老人每日点击「我很好」报平安
5. 系统生成每周健康报告推送给子女

### 流程三：紧急事件响应
1. 系统检测到异常（如跌倒、心率异常）
2. 立即触发多级预警：
   - 第一级：家庭终端声光报警
   - 第二级：推送通知给绑定的子女APP
   - 第三级：通知社区服务中心
3. 子女/社区确认情况，选择：
   - 误报 → 标记关闭
   - 真实异常 → 联系老人 / 上门查看 / 呼叫120
4. 事件记录存档，用于后续分析

### 流程四：预约上门服务
1. 老人通过家庭终端选择「预约服务」
2. 语音说出需求："我想约明天的午餐配送"
3. AI理解意图，显示可选服务和时间段
4. 老人确认预约
5. 订单推送至服务机构
6. 服务人员按时上门，完成后老人确认
7. 子女端同步收到服务记录

## 应用场景

### 场景一：独居老人跌倒及时救助
王奶奶今年78岁，独居在江岸区永清街道。某天凌晨3点，她起夜时不慎滑倒在卫生间。由于摔得较重，她无法起身也无法够到手机。部署在卫生间的毫米波雷达在5秒内检测到跌倒事件，系统立即触发紧急预警。家庭终端响起"检测到您可能摔倒，是否需要帮助"的语音询问，60秒未回应后自动升级为紧急警报。在北京工作的儿子手机APP收到推送，立即视频接入查看情况，同时社区服务中心值班人员收到通知。10分钟内，社区工作人员上门，15分钟后120到达。王奶奶得到及时救治，避免了独自躺在地上数小时的危险。

### 场景二：认知症老人防走失
李爷爷今年82岁，患有轻度阿尔茨海默病，有时会忘记回家的路。家人为他佩戴了带定位功能的智能手环，并在系统中设置了以家为圆心、500米为半径的电子围栏。某天下午，李爷爷独自外出散步，当他走出围栏范围时，系统立即向其女儿发送预警，并在地图上显示实时位置。女儿通过APP远程播放语音引导："爸爸，请原地等一下，我马上来接您"。同时，她通过共享定位功能，引导附近的社区志愿者前往协助。整个过程不到20分钟，李爷爷被安全带回家中。

### 场景三：慢病老人健康管理
张大爷65岁，患有高血压多年。他每天早晚使用智能血压计测量血压，数据自动同步到平台。AI分析发现，张大爷最近一周的晨起血压持续偏高，且有上升趋势。系统向张大爷推送健康建议："您近期血压控制不理想，建议减少盐分摄入，并考虑咨询医生是否需要调整用药"。同时，系统将这一异常趋势同步给张大爷的家庭医生。家庭医生通过平台预约了一次视频问诊，及时调整了用药方案。三周后，张大爷的血压恢复到正常范围。

### 场景四：空巢老人情感陪伴
刘奶奶今年70岁，子女都在外地工作，平时一个人在家经常感到孤独。自从安装了智慧养老终端，她每天都会和AI语音助手"小慧"聊天。小慧会给她讲评书、播放她喜欢的黄梅戏、陪她玩成语接龙游戏，还会在她生日时播放子女提前录制的祝福视频。更重要的是，系统会分析刘奶奶的情绪状态——当检测到她连续几天说话减少、语调低沉时，会主动提醒她的儿子多打电话关心。儿子接到提醒后，当晚就通过视频通话陪母亲聊了一个小时。

### 场景五：养老机构智能巡房
江岸区悦年华颐养中心有100张床位，夜间只有2名护理员值班。传统模式下，护理员每2小时巡房一次，工作强度大且覆盖不全。部署智慧养老系统后，每个房间的智能床垫和监测设备实时采集老人的呼吸、心率、离床状态。护理员通过管理后台的"夜间巡房"功能，一目了然地看到所有老人的状态：绿色表示正常睡眠，黄色表示离床（可能去卫生间），红色表示异常需要关注。系统还会智能提示："302房张奶奶离床已超过15分钟，建议前往查看"。护理员只需重点关注异常提示，大大提高了工作效率，也让老人的安全得到更好保障。

## 技术架构建议

### 数据流架构
```
家庭智能设备 → 边缘计算网关 → 云端数据平台 → AI分析引擎 → 多端推送
      ↓              ↓              ↓              ↓
   本地实时处理    数据清洗聚合    健康档案存储    预警/建议/报告
```

### 关键技术指标
- 跌倒检测响应时间：< 5秒
- 紧急预警送达时间：< 10秒
- 健康数据同步延迟：< 30秒
- 语音识别准确率：> 95%
- 系统可用性：99.9%

### 数据安全要求
- 健康数据加密存储，符合《个人信息保护法》
- 视频数据本地存储为主，云端存储需用户明确授权
- 支持数据导出和删除权利
- 与医疗机构数据对接需符合相关医疗数据法规

## 商业模式建议

- 硬件设备销售：智能网关+传感器套装，定价1999-4999元
