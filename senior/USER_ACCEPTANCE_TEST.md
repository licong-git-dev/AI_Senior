## 技术说明

### RLS策略修复详情
**修复前问题**：
- 错误：PostgreSQL报"infinite recursion detected in policy for relation profiles"
- 原因：RLS策略嵌套查询profiles表导致循环引用
- 症状：前端查询返回HTTP 500错误

**修复方案**：
简化RLS策略，直接比较user_id，避免嵌套查询：
```sql
-- 修复后的策略
CREATE POLICY "用户可查看自己的健康数据" ON health_data FOR SELECT
USING (user_id = auth.uid());
```

**修复时间**：2025-11-17 13:40
