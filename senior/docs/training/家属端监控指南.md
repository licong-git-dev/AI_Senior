# 智慧养老系统 - 家属端监控指南

## 欢迎使用智慧养老家属端APP

本指南将帮助您通过手机APP远程关注和照料家中老人的健康状况，实现智能化养老照护。

---

## 📱 APP下载与安装

### 下载方式
- **Android用户**：在应用商店搜索"智慧养老家属版"
- **iOS用户**：在App Store搜索"智慧养老家属版"
- **扫码下载**：扫描设备附带二维码直接下载

### 安装步骤
1. **下载APP** → 约20MB大小
2. **安装应用** → 按提示完成安装
3. **打开APP** → 首次启动需要注册
4. **登录账号** → 使用家属手机号码注册

### 系统要求
- **Android版本**：7.0以上
- **iOS版本**：11.0以上
- **内存**：至少2GB可用空间
- **网络**：支持WiFi和4G/5G

## 🔐 账户注册与绑定

### 家属账户注册
1. **点击"注册"** → 进入注册页面
2. **输入手机号码** → 接收验证码
3. **设置登录密码** → 至少6位数字或字母
4. **填写身份信息** → 与老人关系、姓名等
5. **同意服务条款** → 点击确认注册

### 绑定老人设备
1. **进入"设备管理"**
2. **点击"添加设备"**
3. **扫描设备二维码** → 位于设备背面
4. **输入设备编号** → 如：SN123456789
5. **设置老人信息** → 姓名、年龄、健康状况
6. **完成绑定** → 确认绑定关系

### 授权管理
- **主管理员** - 您本人，拥有全部权限
- **子管理员** - 可添加其他家属，需您授权
- **查看权限** - 仅能查看数据，无法修改设置
- **紧急联系人** - 有权限接收紧急提醒

## 📊 健康数据监控

### 实时健康数据
**查看方式**：
1. **进入"健康监控"主页**
2. **查看实时数据**：
   - 当前血压值
   - 当前心率
   - 当前体温
   - 活动状态
   - 睡眠质量

**数据解读**：
- **血压正常**：收缩压90-140，舒张压60-90
- **心率正常**：60-100次/分钟
- **体温正常**：36-37.3℃
- **血氧正常**：95-100%

### 历史数据图表

3. **查看图表**：
   - 血压变化曲线
   - 心率变化趋势
   - 体温波动图
   - 用药依从性统计

### 健康报告
1. **进入"健康报告"**
2. **生成报告**：
   - 选择时间范围
   - 选择报告类型
   - 自动生成分析报告
3. **报告内容**：
   - 健康数据汇总

## 🚨 预警与提醒

### 智能预警设置
**预警类型**：
- **健康数据异常** - 血压、心率、体温超过正常范围
- **用药提醒** - 忘记服药或服药时间异常
- **设备异常** - 电量低、网络断开、传感器故障
- **活动异常** - 长时间无活动或异常活动

**预警级别**：
- 🔴 **紧急预警** - 需要立即处理
- 🟡 **重要提醒** - 建议及时关注
- 🔵 **一般提醒** - 日常关注事项

### 预警接收方式
1. **APP推送通知** - 实时接收预警信息
2. **短信提醒** - 重要预警会发送短信
3. **电话通知** - 紧急情况会直接拨打
4. **邮件报告** - 每日健康报告发送邮箱

### 预警响应流程
1. **收到预警** → APP显示预警详情
2. **查看数据** → 点击查看详细健康数据
3. **联系老人** → 直接拨打视频通话
4. **联系护理** → 必要时呼叫专业护理
5. **记录处理** → 在APP中记录处理情况

## ❓ 常见问题解答

### 安装与登录问题

**Q: 忘记登录密码怎么办？**
A: 在登录页面点击"忘记密码"，通过手机验证码重置密码。

**Q: 无法收到验证码？**
A: 检查手机信号，确认手机号正确，或等待1-2分钟后重试。

**Q: APP安装失败？**
A: 检查手机存储空间，确保有足够空间；或重启手机后重试安装。

**Q: 设备绑定失败？**
A: 确认设备已开机且网络连接正常，检查设备编号是否正确输入。

### 功能使用问题

**Q: 健康数据不准确？**
A: 确保设备佩戴正确，传感器清洁，或联系客服进行设备校准。

**Q: 视频通话卡顿？**
A: 检查网络信号强度，建议使用WiFi或4G网络，通话时保持设备稳定。

**Q: 预警通知没有收到？**
A: 检查APP通知权限，确保通知功能开启，或检查勿扰模式设置。

**Q: 用药提醒不准确？**
A: 确认用药时间设置正确，检查系统时间是否准确，或手动校准设备时间。

### 数据与隐私问题

**Q: 健康数据安全吗？**
A: 系统采用银行级加密技术，数据仅供授权用户查看，严格保护用户隐私。

**Q: 可以删除历史数据吗？**
A: 可以删除特定时间段的数据，但为了健康分析建议保留至少3个月数据。

**Q: 第三方能看到数据吗？**
A: 不会向第三方泄露个人数据，除非您主动授权分享给医院或医生。
