# 智慧养老系统 - 老人端操作指南

## 欢迎使用智慧养老系统

本指南将帮助您轻松使用智能护理设备，让您的晚年生活更加安心。

---

## 📱 主界面介绍

### 开机步骤
1. **长按电源键3秒** → 屏幕亮起
2. **点击"开始"按钮** → 进入主菜单
3. **滑动解锁** → 进入日常使用界面

### 主菜单功能
- 🏠 **主页** - 查看今日健康数据
- ❤️ **健康监测** - 血压、心率、体温测量
- 📞 **紧急求助** - 一键呼叫家属和护理人员
- 💊 **用药提醒** - 按时服药提醒
- 📞 **视频通话** - 与家属视频聊天
- ⚙️ **设置** - 个性化设置

---

## ❤️ 健康监测使用指南

### 血压测量
1. **佩戴血压袖带** - 绑在左手臂，位置与心脏同高
2. **点击"血压测量"** - 设备自动充气测量
3. **等待30秒** - 显示测量结果
4. **查看数据** - 屏幕显示收缩压、舒张压、心率

**⚠️ 测量注意事项：**
- 测量前静坐5分钟
- 手臂放置在桌上，手掌向上
- 测量时保持安静，不要说话

### 体温测量
1. **点击"体温测量"**
2. **将体温枪对准额头** - 距离3-5厘米
3. **按一下测量键** - 3秒出结果
4. **查看体温数值** - 正常范围：36-37.3℃

### 心率监测
1. **佩戴智能手表** - 紧贴手腕，不要过松或过紧
2. **点击"心率监测"**
3. **保持静止** - 测量时间约1分钟
4. **查看心率** - 正常范围：60-100次/分钟

## 📞 紧急求助功能

### 🚨 一键求助
1. **找到红色按钮** - 主界面右下角
2. **长按3秒** - 屏幕显示"正在呼叫..."
3. **等待连接** - 自动拨打家属和护理人员电话
4. **说明情况** - 系统会自动告知您的位置

### 📍 自动定位
- 系统会自动获取您的位置信息
- 家属和护理人员能准确找到您
- 无需手动输入地址

### 🔊 语音求助
1. **说出"救命"** - 设备自动识别
2. **或者说出"呼叫家属"**
3. **或者说出"呼叫护理"**
4. **系统会自动拨打对应电话**

---

## 💊 用药提醒设置

### 添加用药提醒
1. **点击"用药提醒"**
2. **点击"添加药物"**
3. **输入药物名称** - 如：降压药
4. **设置服用时间** - 如：每天早8点
5. **选择提醒方式** - 声音+屏幕显示

### 服用药物流程
1. **听到提醒声** - 设备发出提醒
2. **点击"确认服药"**
3. **记录用药情况** - 按时服药/忘记服药
4. **家属可查看记录** - 了解您的用药情况

---

## 📞 视频通话功能

### 拨打家属电话
1. **点击"视频通话"**
2. **选择家属联系人** - 头像显示
3. **点击"呼叫"按钮**
4. **等待接听** - 接通后开始视频

### 接听电话
1. **听到铃声** - 设备会响起铃声
2. **点击"接听"** - 蓝色按钮
3. **开始通话** - 可以看到家属画面
4. **点击"挂断"结束通话**

### 调整音量
- **音量+/-按钮** - 通话时可以调节音量
- **免提模式** - 点击扬声器图标
- **静音模式** - 点击麦克风图标

---

## ⚙️ 个人设置

### 调节屏幕亮度
1. **进入"设置"**
2. **点击"屏幕亮度"**
3. **滑动调节** - 建议设置为中等亮度

### 调节字体大小
1. **进入"设置"**
2. **点击"字体大小"**
3. **选择"大字体"** - 建议老人使用大字体

### 声音设置
1. **进入"设置"**
2. **点击"声音"**
3. **调节音量** - 确保能清楚听到提醒

### 网络连接
1. **进入"设置"**
2. **点击"WiFi"**
3. **选择网络** - 输入密码连接

## 🔋 设备维护

### 充电指南
1. **找到充电口** - 设备底部
2. **插入充电线** - 对准接口轻轻插入
3. **屏幕显示充电图标** - 确认开始充电
4. **充电完成** - 屏幕显示100%

**⚠️ 充电注意事项：**
- 使用原装充电器
- 不要在洗澡时充电
- 充满后及时拔掉充电器

### 日常清洁
- **屏幕清洁** - 用软布轻轻擦拭
- **设备外壳** - 用湿纸巾擦拭
- **传感器清洁** - 保持清洁，避免遮挡

### 设备放置
- **放置位置** - 床头柜或容易触及的地方
- **避免摔落** - 使用防滑垫
- **通风环境** - 避免潮湿和高温

---

## ⚠️ 注意事项

### 使用环境
- **温度范围** - 0℃到40℃
- **湿度范围** - 20%到80%
- **避免水溅** - 不在洗澡时使用
- **避免磁场** - 远离大型电器

### 紧急情况处理
1. **设备无法开机** → 检查充电和电源
2. **测量结果异常** → 联系护理人员确认
3. **求助功能失效** → 直接拨打120或家属电话
4. **网络连接失败** → 重启设备或联系管理员

### 数据隐私
- 您的健康数据仅供您和授权家属查看
- 系统采用加密传输，保护您的隐私
- 不会向第三方泄露您的个人信息
