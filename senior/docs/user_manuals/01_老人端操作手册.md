## ❤️ 详细使用步骤

### 🩺 血压测量（每天建议测量1-2次）

#### 准备工作：
1. **坐在椅子上**，背靠椅背
2. **休息5分钟**，不要说话
3. **左手放在桌上**，手掌向上
4. **脱掉厚衣服**，袖子不要太紧

#### 测量步骤：
1. **点击"健康监测"**
2. **点击"血压测量"**
3. **把袖带绑在左手臂**：
   - 袖带下边缘在肘关节上方2指宽
   - 松紧适中，能伸进一根手指
4. **点击"开始测量"**
5. **袖带会自动充气**，不要紧张
6. **等待30秒**，保持安静
7. **查看结果**，设备会显示：
   - 收缩压（高压）：应该在90-140之间
   - 舒张压（低压）：应该在60-90之间
   - 心率：应该在60-100之间

#### ⚠️ 注意事项：
- 测量前不要喝咖啡或茶
- 不要刚运动完就测量
- 如果结果不正常，休息5分钟再测一次
- 连续3次结果都异常，要联系护理人员

### 🌡️ 体温测量（每天测量1次）

#### 测量步骤：
1. **点击"健康监测"**
2. **点击"体温测量"**
3. **拿起体温枪**（设备附带的小设备）
4. **对准额头**，距离3-5厘米
5. **按一下测量键**，会听到"滴"声
6. **等待3秒**，看屏幕显示结果
7. **正常体温**：36-37.3℃

#### ⚠️ 注意事项：
- 测量前不要刚洗完脸
- 测量时保持头部不动
- 如果体温超过37.5℃，要联系护理人员

### ⏰ 心率监测（每天建议测量1-2次）

#### 测量步骤：
1. **佩戴智能手表**：
   - 戴在左手腕
   - 紧贴皮肤，不要太松
   - 屏幕朝向自己
2. **点击"健康监测"**
3. **点击"心率监测"**
4. **保持安静**，不要说话
5. **等待1分钟**，设备会自动测量
6. **查看结果**：正常心率60-100次/分钟

## 🚨 紧急求助功能

### 这是最重要的功能！请认真学会：

#### 什么情况下需要求助：
- 感觉胸闷、胸痛
- 感觉头晕、头痛严重
- 感觉呼吸困难
- 感觉恶心想吐
- 摔倒了但爬不起来
- 感到非常害怕或不安
- 其他任何让您觉得危险的情况

#### 如何使用紧急求助：
1. **找到红色按钮**（主界面右下角）
2. **长按3秒钟**，屏幕会显示"正在呼叫..."
3. **不要松手**，继续按住
4. **等待连接**，设备会自动：
   - 给家属打电话
   - 给护理人员打电话
   - 说明您的位置
5. **接通后**：说明您的情况，如果说不出话，设备会替您说明

#### 🔊 语音求助（更方便）：
您也可以说话求助：
- **说"救命"** → 设备自动识别
- **说"呼叫家属"** → 给家属打电话
- **说"呼叫护理"** → 给护理人员打电话

#### 📍 自动定位功能：
- 设备会自动获取您的位置
- 家属和护理人员能准确找到您
- 如果您说不清楚在哪里，设备会帮忙说明位置

## 💊 用药管理

### 设置用药提醒：

#### 添加药物：
1. **点击"用药提醒"**
2. **点击"添加药物"**
3. **输入药物名称**（可以用语音）：
   - 如：降压药、心脏病药
4. **选择服用时间**：
   - 早：6:00-10:00
   - 中：11:00-15:00  
   - 晚：16:00-22:00
5. **设置提醒方式**：
   - 声音提醒（建议选择）
   - 屏幕闪烁
6. **点击"确认"**

#### 服药流程：
1. **听到提醒声** → 设备发出"叮咚"声
2. **点击"确认服药"** → 屏幕上会出现按钮
3. **服用药物** → 按医生说的服用
4. **再次点击"确认"** → 告诉系统您已经服药

#### 📊 查看用药情况：
- **点击"用药记录"**
- **查看今天** → 今天的服药情况
- **查看本周** → 这星期的服药情况
- **家属可以看到** → 他们知道您有没有按时服药

## 📞 视频通话

### 给家属打电话：

#### 步骤：
1. **点击"视频通话"**
2. **选择家属**：
   - 儿子/女儿的头像
   - 其他家属的头像
3. **点击"呼叫"**（绿色按钮）
4. **等待接听**：
   - 会听到铃声
   - 屏幕显示家属的画面
5. **开始聊天**：
   - 您可以看到家属
   - 可以和家属说话
   - 家属也可以看到您

### 接听家属电话：
1. **听到铃声** → 设备会响起
2. **点击"接听"**（蓝色按钮）
3. **开始通话** → 看到家属的画面
4. **通话中**：
   - 说话声音大一点，设备音量调大
   - 如果听不清，点击"免提"
   - 如果想静音，点击麦克风图标
5. **结束通话** → 点击红色的"挂断"按钮

### 调节音量：
- **声音小** → 点击"音量+"
- **声音大** → 点击"音量-"
- **免提模式** → 点击扬声器图标
- **静音模式** → 点击麦克风图标

1. **点击"设置"**
2. **点击"网络"**

## 🔋 设备维护

### 充电指南：

#### 充电步骤：
1. **找到充电口** → 设备底部的小口
2. **插入充电线** → 对准接口轻轻插入
3. **确认充电** → 屏幕显示充电图标和闪电符号
4. **充电时间** → 约2-3小时充满
5. **充满显示** → 屏幕显示100%

#### ⚠️ 充电注意事项：
- **使用原装充电器**（白色小盒子）
- **不要在洗澡时充电**
- **充满后及时拔掉**充电器
- **充电时不要用设备**测量血压等

#### 电池电量提醒：
- **电量低于20%** → 屏幕提示"请充电"
- **电量低于10%** → 红色提醒，必须充电
- **电量0%** → 设备会自动关机

### 日常清洁：
- **屏幕清洁** → 用干净软布轻轻擦拭
- **设备外壳** → 用微湿的纸巾擦拭
- **传感器清洁** → 保持清洁，避免遮挡
- **袖带清洁** → 用湿布擦拭，晾干再用

### 设备放置：
- **放置位置** → 床头柜或容易够到的地方
- **避免摔落** → 使用防滑垫放在设备下面
- **通风环境** → 放在干燥、通风的地方
- **避免潮湿** → 不要放在洗手间等潮湿地方

## ⚠️ 重要注意事项

### 使用环境要求：
- **温度范围** → 0℃到40℃
- **湿度范围** → 20%到80%
- **避免水溅** → 不要在洗澡时使用
- **避免磁场** → 远离大型电器、微波炉

### 紧急情况处理：

#### 设备无法开机：
1. **检查充电** → 设备可能没电了
2. **长按电源键10秒** → 强制重启
3. **如果还是不开机** → 联系技术支持

#### 测量结果异常：
1. **重新测量一次** → 休息5分钟后再测
2. **检查测量方法** → 是否按要求操作
3. **如果仍然异常** → 联系护理人员确认

#### 求助功能失效：
1. **直接拨打120** → 生命安全问题优先
2. **给家属打电话** → 用手机直接拨打
3. **联系邻居帮忙** → 如果能说话的话

#### 网络连接失败：
1. **重启设备** → 长按电源键重启
2. **检查WiFi** → 确认家庭网络正常
3. **联系管理员** → 请技术人员帮忙

### 数据隐私保护：
- **您的健康数据很安全** → 银行级加密保护
- **只有您和家属能看** → 不会泄露给其他人
- **医院可以看到** → 只有您同意授权的医生才能看
