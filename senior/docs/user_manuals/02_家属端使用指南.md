## 🚨 智能预警系统

### 预警分类与级别

#### 🔴 **一级预警（紧急）**
**触发条件**：
- 血压严重异常（>180/<80 或 >120/<40）
- 心率严重异常（>120/<40次/分钟）
- 体温异常（>39℃或<35℃）
- 血氧异常（<85%）
- 老人主动SOS求助信号
- 长时间无活动（>6小时）

**响应要求**：**5分钟内必须处理**

#### 🟡 **二级预警（重要）**
**触发条件**：
- 血压轻度异常（140-160/90-100）
- 心率轻度异常（110-120次/分钟）
- 体温轻度异常（37.5-38.5℃）
- 用药依从性差（连续2次漏服）
- 设备电量严重不足（<10%）

**响应要求**：**30分钟内处理**

#### 🔵 **三级预警（一般）**
**触发条件**：
- 数据波动提醒
- 定期检查提醒
- 设备维护提醒
- 用药库存不足
- 网络连接异常

**响应要求**：**2小时内处理**

## 🔧 故障排除与技术支持

### 常见问题解决

#### 安装与登录问题：

**Q1：APP安装失败怎么办？**
A：解决方案：
1. 检查手机存储空间是否充足（至少2GB）
2. 重启手机后重新安装
3. 检查网络连接是否正常
4. 联系客服获取最新版本安装包

**Q2：忘记登录密码怎么办？**
A：解决方案：
1. 在登录页面点击"忘记密码"
2. 输入注册手机号码
3. 接收并输入验证码
4. 设置新密码并登录
5. 如无法收到验证码，联系客服重置

**Q3：验证码收不到？**
A：解决方案：
1. 检查手机信号是否正常
2. 确认手机号码输入正确
3. 等待1-2分钟后重新请求
4. 检查是否被手机安全软件拦截
5. 联系运营商查询短信

#### 功能使用问题：

**Q4：视频通话画面卡顿？**
A：解决方案：
1. 检查网络信号强度
2. 切换到WiFi网络
3. 关闭其他占用网络的应用
4. 重启APP和路由器
5. 调整视频质量设置

**Q5：健康数据不准确？**
A：解决方案：
1. 检查设备佩戴是否正确
2. 确保传感器清洁
3. 重新校准设备参数
4. 对比备用测量设备
5. 联系技术支持设备检测

**Q6：预警通知没有收到？**
A：解决方案：
1. 检查APP通知权限设置
2. 确认勿扰模式是否开启
3. 检查通知提醒设置
4. 重启手机后测试
5. 联系客服检查通知配置

#### 设备相关问题：

**Q7：设备离线无法连接？**
A：解决方案：
1. 检查设备电源和充电状态
2. 重启设备和路由器
3. 检查WiFi密码是否正确
4. 靠近路由器重新连接
5. 联系技术支持现场检查

**Q8：设备测量数据异常？**
A：解决方案：
1. 确认老人身体状态正常
2. 检查测量环境和姿势
3. 重新校准设备
4. 使用备用设备对比
5. 联系医生确认数据
