## 故障排除

### 常见问题

1. **依赖安装失败**
   ```bash
   # 清理缓存重新安装
   rm -rf node_modules package-lock.json
   npm install
   ```

2. **构建错误**
   ```bash
   # 检查TypeScript类型错误
   npm run type-check
   
   # 检查代码规范
   npm run lint
   ```

3. **Supabase连接失败**
   - 检查环境变量配置
   - 验证API密钥有效性
   - 确认数据库策略配置
